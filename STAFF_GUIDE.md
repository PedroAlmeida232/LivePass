# 📘 Guia de Operação — Equipe Staff (Scanner)

Este guia orienta o uso do aplicativo de validação de ingressos durante o evento.

---

## 1. Acesso ao Scanner
1. Realize o login com sua conta de Staff em `/login`.
2. No menu, acesse a opção **"Validar Ingressos"** (ou vá direto para `/scanner`).
3. Autorize o uso da câmera no seu navegador.

---

## 2. Como Validar um Ingresso
1. Peça para o cliente abrir o aplicativo LivePass e acessar o ingresso.
2. Certifique-se de que o **brilho da tela do cliente está alto**.
3. Aponte sua câmera para o QR Code dinâmico do cliente.
4. Aguarde o feedback visual e sonoro:

### ✅ Feedback Verde (Sucesso)
*   **Ação:** Libere a entrada do cliente.
*   **Significado:** Ingresso é autêntico, foi pago e nunca foi usado.

### ❌ Feedback Vermelho (Erro)
*   **Cenários Comuns:**
    *   **"Ingresso já utilizado":** O cliente já entrou ou tentou usar o mesmo ingresso em outro portão. **Ação:** Barra a entrada.
    *   **"Token Inválido/Expirado":** O QR Code na tela do cliente mudou. **Ação:** Peça para o cliente aguardar a próxima atualização (máximo 30s) e tente novamente.
    *   **"Pagamento não confirmado":** O cliente ainda não teve o PIX processado. **Ação:** Encaminhar ao suporte/bilheteria.

---

## 3. Dicas de Resolução de Problemas
*   **Câmera não abre:** Recarregue a página ou verifique as permissões de privacidade do site.
*   **QR Code não lê:** Peça para o cliente aumentar o brilho ou afaste um pouco o celular para focar melhor.
*   **Sem internet:** O scanner depende de conexão estável. Se o sinal estiver ruim, tente mudar de posição ou conectar ao Wi-Fi do staff.

---

## 4. Segurança
*   Nunca compartilhe sua senha de Staff.
*   Cada validação é registrada com seu nome e horário para fins de auditoria.
